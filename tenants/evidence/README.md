# Evidence por tenant

Esta pasta guarda evidencias operacionais datadas por tenant.

O que entra aqui:

- smoke tests datados por tenant
- print ou comprovante do owner criado
- evidencia de go-live
- evidencia de restore drill por tenant

Regra pratica:

- cada tenant deve ter sua propria subpasta
- cada evidencia deve ter data no nome do arquivo
- nunca registrar segredo real em markdown
