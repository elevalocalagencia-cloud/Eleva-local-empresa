# Signed Legal Artifacts

Esta pasta e a referencia operacional para contratos assinados por cliente.

## O que vive aqui

- comprovantes ou placeholders operacionais de onde estao os assinados
- nunca segredos
- nunca anexos sensiveis de cliente real no repo publico

## Naming recomendado

`{tenant_id}-{documento}-{YYYY-MM-DD}.pdf`

Exemplos:

- `cli-acme-nda-2026-04-18.pdf`
- `cli-acme-psa-2026-04-18.pdf`
- `cli-acme-dpa-2026-04-18.pdf`

## Politica

- PDFs e DOCX de cliente real NAO devem ir para o repo publico
- usar esta pasta apenas como ponto de convencao e documentacao operacional
- se houver subpastas por tenant, manter o mesmo padrao de naming
